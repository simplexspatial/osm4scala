osm4scala

Scala library and Spark Polyglot connector for OpenStreetMap Pbf files

Get Started

Scala library

Forget about the OSM Pbf format complexity and think about a simple iterators of primitives (nodes, ways and relations) or blob blocks.
Easy and high performance library, with millions of entities processed per second, in only one thread.

Spark Polyglot

Creating DataFrames representing OSM Pbf file contents from PySpark, Spark Scala, Spark SQL or SparkR is as simple as any other format file.

Community / Commercial support

Two support ways:

  • Free support, via StackOverflow.
  • Commercial support, via the Author services.
